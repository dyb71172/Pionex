### 最受欢迎的几个机器人

**网格交易机器人(Grid Trading Bot)**

这是派网最经典的工具。原理很简单——在设定的价格区间内,自动低买高卖。

比如你觉得BTC会在28,000到32,000美元之间震荡,就设置一个网格。机器人会在这个区间内自动挂单,跌了买,涨了卖,不断套利。

我自己试过用网格交易ETH,设置了50个网格,在横盘行情里一个月下来收益率大概7%左右。当然这个收益不是每次都有,要看行情配合。

**无限网格机器人(Infinity Grid)**

这个是网格的升级版。普通网格有上下限,如果价格突破了你设定的区间就停止工作。但无限网格没有上限,理论上可以一直跟着上涨趋势吃利润。

适合看好某个币种长期走势,又想在过程中不断套利的情况。

**期现套利机器人(Spot-Futures Arbitrage)**

这个比较稳健,年化收益一般在15%-50%之间。原理是同时持有现货和做空合约,赚取资金费率的差价。

风险相对较低,因为做多和做空对冲了,基本不受币价涨跌影响。我身边有朋友专门用这个,把它当成"加密货币定存",收益比传统理财高多了。

**定投机器人(DCA Bot)**

这个就是定期定额买入,帮你平均成本。可以设置每天、每周或者每月自动买入,完全不用盯盘。

特别适合长期看好某个币种,但又怕买在高点的人。

**屯币宝(HODL)**

这个更狠,专门为长期持有者设计。设置一个目标价位,在到达之前一直定投,到了就停止。

比如你认为BTC会涨到10万美元,就设置这个目标,机器人会帮你在过程中不断囤币。

其他还有追踪止盈止损机器人、杠杆网格、反向网格等等,基本上你能想到的策略都有对应的工具。
